Ejemplo de Applet con Java

Un applet conforme a lo que encontramos en las paginas de Oracle: "es una aplicación hecho en Java que se llama desde un documento HTML y se ejecula dentro del navegador " Es necesario recordar que el navegador debe tener instalado el plugin de la Máquina Virtual de Java (JVM).


Máquina Virtual de Java Es e programa que se encarga de ejecutar el bytecode generado por el compilador de Java.


Para entender la forma e que se manejaba los applets vamos a ver un ejemplo que se presenta el clásico Hola mundo :


Como puede observarse la única variable que utiliza este applet es g que reprenta un objeto de la clase Graphics sobre el cual vamos a dibuar la cadena de texto que resulta de ejecutar el applet.


Para ejecutarlo es necesario mandarlo llamar desde una página HTML utilizando la etiqueta < applet > o la etiqueta < object >. En el primer caso el archivo .class se declara en el atributo code, en el segundo se delcara un elemento param donde sus atributos name será code y value será el nombre del applet .Cabe mencionar que en el segundo caso es necesario declarar otros elementos param adicionales y en elemento object declarar algunos parámetros como classid, codebase / y type. en ambos casos es necesario declarar el tamaño de la ventana del applet con los parámetros width y height.


La salida del programa será: Hola mundo!!
